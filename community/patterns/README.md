## What is a Pattern?

A pattern is a reusable solution to a common problem in AI coding. It includes:
- The problem it solves
- The solution approach
- Code examples
- When to use it (and when not to)

## How to Submit

1. Create a markdown file following the pattern template
2. Include clear examples and use cases
3. Open a PR

See [CONTRIBUTING.md](../../CONTRIBUTING.md) for the pattern template.
